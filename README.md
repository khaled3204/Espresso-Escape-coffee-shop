# Espresso Escape Coffee Shop Website

# Goal of the Project: 

# &nbsp;The goal of this project was to create a visually appealing and functional website for "Espresso Escape Coffee Shop." The website provides customers with an intuitive navigation experience, detailed information about coffee options, and a means to contact the shop.The website emphasizes aesthetic design with user-friendly functionality to enhance customer engagement.

# 

# Key Parts of the Code:

# 1)Home Page (HOME.html)

# Purpose: Serves as the welcoming page for visitors.

# Features:

# -A responsive background image featuring a coffee theme.- A welcoming message styled with large font sizes and text shadows for better visibility.

# \- Navigation links to key pages: Home, Menu, and Contact Us.

# \- A brief description of the coffee shop's ambiance and services,styled within a semi-transparent black container to ensure readability over the background.

# \- Persistent footer with copyright information.

# 2)Coffee Menu (coffee\_menu.html)

# Purpose: Displays a detailed coffee menu with descriptions and

# Links to individual coffee detail pages. 

# Features:

# \- A table format for organizing coffee types, descriptions, and prices.

# \- Links to detailed pages for each coffee type (e.g., Espresso.html,Latte.html).

# \- Custom hover effects for navigation links and clickable coffee names.

# \- Responsive design ensuring the table remains visually appealing on different screen sizes. 3)Coffee Detail Pages: 

# Each coffee detail page (e.g.,Affogato.html, Latte.html,Espresso.html, etc.) follows a consistent structure:

# -card Design: Each page uses a card layout with rounded corners, shadows, and semi-transparent backgrounds for a modern appearance.

# Content: 

# \- A bold header with the coffee name.

# \- A descriptive paragraph detailing the coffee.

# \- A list of ingredients styled for readability.

# \- A navigation link to return to the main menu.

# 4)Individual Coffee Pages:

# \- Affogato (Affogato.html): Scoop of vanilla ice cream topped  with hot espresso.

# \- Americano (Americano.html): Espresso with hot water for a milder taste.

# \- Cappuccino (Cappuccino.html): Espresso, steamed milk, and foam in harmony.

# \- Cold Brew (Cold\_brew.html): Smooth, chilled coffee steeped for hours.

# \- Espresso (Espresso.html): Strong and bold, a coffee lover's classic.

# \- Flat White (Flat\_White.html): Velvety steamed milk over a double shot of espresso.

# \- Latte (Latte.html): Rich espresso with steamed milk and a small foam layer.

# \- Macchiato (Macchiato.html): Espresso with a dollop of steamed milk foam.

# \- Mocha (Mocha.html): Espresso with chocolate and steamed milk.

# \- Nitro Cold Brew (Nitro\_Cold\_Brew.html): Cold brew infused with nitrogen for a creamy texture.

# \- Turkish Coffee (Turkish\_Coffee.html): Strong, rich, unfiltered coffee with optional cardamom.

# \- Water (Water.html): A fancy water served in a glass bottle.

# 5)Contact Page (contact-us.html)

# Purpose: Allows customers to send inquiries or feedback.

# Features:

# \- A form with fields for name, email, and message.

# \- Styled input fields and a submit button with hover effects.

# \- Semi-transparent form background to maintain readability over the coffee-themed background image.

# 6)CSS Styling (Embedded in Each HTML File):-

# Background: Coffee-themed image with responsive scaling, fixed attachment, and no-repeat properties:

# a)Navigation:

# \- Bold, hover-underlined links styled in white for a cohesive look.

# \- Centered and spaced evenly for accessibility.

# b)Cards:

# \- Rounded corners, shadows, and semi-transparent backgrounds for a modern appearance.

# \- Footer:- Fixed at the bottom for consistent accessibility and branding.

# Other Details:

# 1)Responsive Design:

# \- Ensured compatibility across devices by using flexible width percentages and media-friendly elements.

# 2)Accessibility:

# -Clear navigation structure and descriptive alt texts for all images to enhance accessibility.

# 3)Visual Consistency:

# \- A unified color palette and font choices to ensure brand consistency across all pages.

# 4)Scalability:

# \- Modular page structure enables easy addition of new menu items or updates to existing pages.

# 5)Code Comments:

# \- Added comments to enhance code readability and provide context for key design choices.
